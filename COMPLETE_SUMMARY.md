## 🚀 HOW TO USE

### **1. Start Server**
```bash
python app.py
```

### **2. Send Symptom Request**
```bash
curl -X POST http://localhost:5000/api/diagnose \
  -F "symptoms=fever and cough" \
  -F "severity=moderate"
```

### **3. Get Detailed Response**
Receives complete JSON with:
- ✅ Disease identified with confidence %
- ✅ All OTC medications with dosages
- ✅ All prescription alternatives
- ✅ Brand names (Indian brands)
- ✅ Frequency information
- ✅ When to see doctor
- ✅ Severity level
- ✅ Duration estimate
